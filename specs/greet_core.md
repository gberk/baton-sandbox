# greet CLI — Core Commands

## hello

`greet hello <name>` prints a greeting to stdout:

```
Hello, <name>!
```

The name is required. If omitted, the command prints a usage error.

## goodbye

`greet goodbye <name>` prints a farewell to stdout:

```
Goodbye, <name>!
```

The name is required. If omitted, the command prints a usage error.
